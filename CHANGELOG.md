## [0.1.0] - 2025-01-11

### Added
- Initial release
- Support for Ollama, vLLM, and LMStudio backends
- Core performance metrics (TTFT, TPOT, Throughput)
- SSH-based remote testing
- JSON configuration system
- Basic visualization with matplotlib/seaborn
- Streaming and non-streaming mode support
- Automatic service health checking
- Result export in JSON format

### Documentation
- README with quick start guide
- Configuration examples
- API documentation
- Contributing guidelines

### Known Issues
- Limited error recovery in SSH connections
- Visualization could be more customizable
- No support for concurrent request testing yet
